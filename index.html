Algunos lenguajes de la web

HTML + CSS + Javascript

  • HTML5

    • HTML5(Hiper text Markup Languaje, Version 5) es la quinta revision importante del lenguaje basico de la Word Wilde Web, HTML.

    Usa la extension.html

  • CSS

    • Hoja de estilo en cascada o CSS (siglas en ingles de cascading style sheets) es un lenguaje para definir y crear la presentacion de un documento estructurado escrito en HTML.

    Usa la extencion css

  • JS

    • JavaScript(abreviado comunmente"JS") es un lenguaje de programacion interpretado utilizado principalmente en el lado del cliente (client-side), implementando como parte de un navegador Web permitiendo mejoras en la interfaz de usuario y paginas web dinamicas

    Usa la extension .js
